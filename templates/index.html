Heun's Method

Improved Euler Method · Predictor-Corrector

Short Discussion

Heun’s Method is an improved version of Euler’s Method used to solve ordinary differential equations numerically. Instead of using only one slope, it averages two slopes to obtain a more accurate approximation.


Given:

\[ \frac{dy}{dx}=f(x,y), \quad y(x_0)=y_0 \]

Predictor Formula:

\[ \tilde y_{i+1}=y_i+h f(x_i,y_i) \]

Corrector Formula:

\[ y_{i+1}=y_i+\frac{h}{2} [f(x_i,y_i)+f(x_{i+1},\tilde y_{i+1})] \]

✨ The method reduces local truncation error and provides better stability than standard Euler.

Worked Examples

Example 1 · Step-by-Step Solution

Solve using Heun's method:

\[ \frac{dy}{dx}=x+y, \quad y(0)=1, \quad h=0.1 \]

Goal: approximate \(y(0.1)\) with one complete Heun iteration.

📌 Step 1 — Identify function & initial data

\(f(x,y) = x + y\), \(x_0 = 0\), \(y_0 = 1\), step size \(h = 0.1\). Then \(x_1 = x_0 + h = 0.1\).

📌 Step 2 — Compute first slope (at start point)

\[ f(x_0, y_0) = f(0,1) = 0 + 1 = 1 \]

📌 Step 3 — Predictor (Euler forward estimate)

\(\tilde y_{1} = y_0 + h \cdot f(x_0, y_0)\)

\[ \tilde y_{1} = 1 + 0.1 \times (1) = 1 + 0.1 = 1.1 \]

Predicted value: \(\tilde y_1 = 1.1\) at \(x_1 = 0.1\).

📌 Step 4 — Second slope (using predicted endpoint)

Evaluate \(f\) at \((x_1, \tilde y_1) = (0.1,\; 1.1)\):

\[ f(0.1, 1.1) = 0.1 + 1.1 = 1.2 \]

📌 Step 5 — Corrector (average of slopes)

Heun's corrector formula:

\[ y_{1} = y_0 + \frac{h}{2}\big[ f(x_0, y_0) + f(x_1, \tilde y_1) \big] \]

Substitute values:

\[ y_{1} = 1 + \frac{0.1}{2} \big[ 1 + 1.2 \big] = 1 + 0.05 \times (2.2) = 1 + 0.11 = 1.11 \]
🎯 Final Approximation: \[ y(0.1) \approx 1.11 \]

Exact solution: \(y(x)=2e^{x}-x-1\) ⇒ \(y(0.1)\approx 1.11034\), error ≈ 0.00034.

✅ Insight: The predictor gave 1.1, and the corrector refined the value to 1.11. Averaging slopes captures curvature better, yielding high accuracy even with a single step.

Example 2 · Detailed Steps

Apply Heun's method to the ODE:

\[ \frac{dy}{dx} = -2xy, \quad y(0)=1, \quad h=0.2 \]

We compute \(y(0.2)\) using one Heun step.

📌 Step 1 — Problem setup

\(f(x,y) = -2xy\), initial: \(x_0 = 0\), \(y_0 = 1\), step \(h = 0.2\). Then \(x_1 = 0.2\).

📌 Step 2 — First slope at \((x_0, y_0)\)

\[ f(0, 1) = -2 \cdot 0 \cdot 1 = 0 \]

Derivative is zero initially; Euler predictor will initially remain flat.

📌 Step 3 — Predictor (Euler guess)

\[ \tilde y_{1} = y_0 + h \cdot f(x_0, y_0) = 1 + 0.2 \times 0 = 1 \]

Predicted value: \(\tilde y_{1} = 1.0\) at \(x_1 = 0.2\).

📌 Step 4 — Second slope at predicted endpoint

\((x_1, \tilde y_1) = (0.2, 1.0)\)

\[ f(0.2, 1.0) = -2 \times 0.2 \times 1.0 = -0.4 \]

Slope becomes negative due to increase in \(x\).

📌 Step 5 — Heun's corrector (average slope)

\[ y_{1} = y_0 + \frac{h}{2}\big[ f(x_0, y_0) + f(x_1, \tilde y_1) \big] \]

Plug numbers:

\[ y_{1} = 1 + \frac{0.2}{2} \big[ 0 + (-0.4) \big] = 1 + 0.1 \times (-0.4) = 1 - 0.04 = 0.96 \]
✅ Heun's approximation: \[ y(0.2) \approx 0.96 \]

Exact solution: \(y(x) = e^{-x^2}\) ⇒ \(y(0.2)=e^{-0.04}\approx 0.960789\). Error ≈ \(-0.000789\) (very small!).

💡 Insight: With \(h=0.2\), plain Euler would give \(y_1 = 1\) (no change, error ~0.039). Heun's corrector uses the negative slope at the end, dropping the value to 0.96, closely matching the exact solution.
